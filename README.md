 Week 2 - Grade Calculator

# Project Description
This Python program reads student marks from a file,
calculates total, average, and assigns grades.

# Features
- File handling
- Functions
- Loops
- Conditional statements
- Error handling

# How to Run
1. Make sure Python is installed
2. Place test_students.txt in the same folder
3. Run:

   python grade_calculator.py
